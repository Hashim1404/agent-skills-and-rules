# Antigravity Agent Workflow Kit 🤖

Welcome to the unified **Antigravity Agent Workflow Kit** repository! This repository serves as the single source of truth for all specialized capabilities, operational rules, workflows, and best practices designed to empower and structure autonomous AI agents within your workspace.

## 📂 Repository Structure

All components have been carefully combined, connected, and professionally structured to ensure seamless agent operations:

- **`skills/`**: Contains all agent skills (from basic web development to advanced architecture, devops, and design). Each folder includes an instructional `SKILL.md` along with any necessary scripts or examples.
- **`rules/`**: Contains core operational protocols and behavioral directives. For example, `RULE_BOOK.md` establishes fundamental rules like strict visual branding, database synchronization, and guidelines for teaching non-technical builders.
- **`workflows/`**: Houses defined, multi-step agent processes and automated sequences.
- **`shared/`**: Contains cross-cutting utilities and data shared across various skills and agents (e.g., deployment guides, security data, UI/UX scripts).
- **`ARCHITECTURE.md`**: Provides high-level documentation on the system architecture and agent design patterns.

## 🚀 How to Use

These capabilities are designed to be imported and activated by your AI agents to immediately inherit deep contextual knowledge and standardized rule sets. 

By keeping everything consolidated in this `.agents` folder, you ensure that every agent interaction draws from the exact same library of expert configurations, leading to professional, accurate, and highly consistent project outcomes.
